# 云端版本介绍

## 概述

InfoPigu / Cognitive Delta 当前以本地单机版本为主，云端版本正在规划中。

云端版本的方向，不是替代本地版，而是在保持“信息抽取、去重、知识沉淀”核心能力的前提下，进一步扩展容量、协同和跨端能力。

## 云端版本（即将推出）与本地单机版本对比

| 对比项 | 云端版本（即将推出） | 本地单机版本 |
| --- | --- | --- |
| 知识库容量 | 理论上无限制，可持续扩展 | 受本机存储与浏览器本地数据限制 |
| 向量数据库 | 使用更专业、更高效的向量数据库 | 以本地存储为主，能力更轻量 |
| 多端同步 | 支持多端共享同一知识库 | 默认仅当前浏览器、本机可用 |
| 使用方式 | 网页端应用，可跨设备访问 | 本地浏览器插件为主 |
| 外部应用同步 | 可自动同步到外部应用，例如 Notion | 通常需要手动导出或自行处理 |
| 隐私路径 | 需要经过云端服务进行同步和管理 | 数据优先保存在本地，不通过中间服务器 |
| 适用场景 | 团队协作、长期积累、大规模知识管理 | 个人深度阅读、强隐私、本地优先 |

## 云端版本的核心优势

- 知识库文章数容量几乎不受限制，适合长期持续积累。
- 采用更加专业高效的向量数据库，检索、聚类、召回能力更强。
- 支持多端共享知识库，在不同浏览器、不同设备之间保持一致。
- 提供网页端应用，不再局限于单一浏览器插件入口。
- 可以和外部应用自动同步，例如 Notion，方便进入个人知识管理工作流。

## 本地单机版本的核心优势

- 绝对隐私保护是第一优势。
- 数据默认保存在本地，不通过中间服务器。
- 更适合对隐私、敏感阅读记录、本地自治要求较高的用户。

## 建议如何选择

- 如果你更关心隐私、安全、离线掌控，优先选择本地单机版本。
- 如果你更关心容量、跨端同步、知识库协同和外部系统联动，可以期待云端版本。
- 两者未来更适合形成互补：本地版负责隐私优先，云端版负责规模化与协同化。
