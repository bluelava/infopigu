# Cloud Edition Overview

## Overview

InfoPigu / Cognitive Delta is currently centered on a local single-device edition, while a cloud edition is being planned.

The cloud edition is not meant to replace the local version. Its goal is to extend the same core workflow of extraction, deduplication, and knowledge accumulation with more scale, sharing, and cross-device access.

## Cloud Edition (Coming Soon) vs. Local Standalone Edition

| Dimension | Cloud Edition (Coming Soon) | Local Standalone Edition |
| --- | --- | --- |
| Knowledge base capacity | Effectively unlimited and scalable | Limited by local storage and browser data constraints |
| Vector database | Uses a more professional and efficient vector database | Primarily local storage with a lighter capability profile |
| Multi-device sync | Shared knowledge base across devices | Usually available only in the current browser on the current machine |
| Access model | Web application, accessible across devices | Mainly a local browser extension |
| External app sync | Can sync automatically with external apps such as Notion | Usually requires manual export or custom handling |
| Privacy path | Requires cloud services for sync and management | Data stays local by default and does not pass through an intermediate server |
| Best fit | Team collaboration, long-term accumulation, large-scale knowledge management | Personal deep reading, strong privacy, local-first workflows |

## Core Advantages of the Cloud Edition

- Near-unlimited article capacity for long-term knowledge accumulation.
- A more professional and efficient vector database for retrieval, clustering, and recall.
- Shared knowledge base across devices and browsers.
- A web application experience instead of being limited to a single browser extension.
- Automatic synchronization with external tools such as Notion.

## Core Advantages of the Local Standalone Edition

- Absolute privacy protection is the primary strength.
- Data is stored locally by default and does not pass through an intermediate server.
- Better suited to users who care deeply about privacy, sensitive reading history, and local control.

## How to Choose

- Choose the local standalone edition if privacy, security, and offline control matter most.
- Choose the future cloud edition if you care more about scale, sync, collaboration, and external integrations.
- In the long run, the two editions are best seen as complementary: local for privacy-first usage, cloud for scale and collaboration.
